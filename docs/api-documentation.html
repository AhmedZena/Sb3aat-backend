Auth API Documentation

Register User

Method: POST
Registers a new user in the system.
Endpoint: /api/auth/register

Request Body:

{
      "username": "String (required, min length: 2, max length: 50)",
      "email": "String (required, email format, min length: 5, max length: 50)",
      "password": "String (required, min length: 8)",
      "role": "String (optional, values: 'user', 'admin', 'freelancer')"
    }

Response:

{
      "success": true,
      "data": { /* User data */ }
      // Or appropriate error message
    }

Login User

Method: POST
Authenticates a user and returns a token.
Endpoint: /api/auth/login

Request Body:

{
      "email": "String (required, email format, min length: 5, max length: 50)",
      "password": "String (required, min length: 8)"
    }

Response:

{
      "token": "String (Auth Token)"
      // Or appropriate error message
    }
